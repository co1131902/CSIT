XML and JavaScript

What is XML?

      XML stands for eXtensible Markup language and is a markup language much like HTML. XML was designed to store and transport data and, was also designed to be self descriptive.

How is XML used?

      XML is used in many aspects of Web Development and is often used to separate data from presentation. XML is used to complement HTML. Data displayed on HTML should not have to be edited if the data is modified. By using XML that data can be stored in a separate file sheet and by using JavaScript code, you can read the XML file and update the data content of any HTML page.

What is JavaScript?

      JavaScript is an object based client side scripting programming language that is interpreted by a web browser. JavaScript is a versatile language that can be used to perform a wide range of tasks, from simple form validation to complex web applications.

How is JavaScript related to HTML and CSS?

      JavaScript is used to make web pages more dynamic and interactive. It is used to add functionality to HTML elements, modify CSS styles, and interact with data from web servers. JavaScript is included in an HTML file using a <script> tag, which allows the browser to interpret and execute the JavaScript code. Together, HTML, CSS, and JavaScript form the foundation of modern web development, allowing developers to create rich and engaging web applications that run in a browser environment.

What are some common JavaScript elements, properties, methods and functions?
